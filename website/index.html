Why XOLERIC?

🧠

Integrated AI Core

XAI daemon runs natively in the OS, providing instant AI responses without external dependencies. Optimized for CPU inference with 8.9x speedup.

⚡

Custom Rust Microkernel

Minimal 27KB kernel written in Rust with no_std architecture. Designed for security, speed, and modern memory safety.

📦

Native Package Manager

xpkg - custom package manager with AI-assisted dependency resolution. Simple .xpkg format with built-in conflict detection.

🔄

Smart Init System

xinit - lightweight init daemon with self-healing capabilities. Parallel service startup with AI-powered boot prediction.

🛡️

AI-Enhanced Security

Kernel modules with AI anomaly detection. XSHIELD monitors system calls, XNET analyzes network traffic for threats.

🎯

Intelligent Scheduler

XSCHED uses AI to predict optimal process scheduling. Considers CPU affinity, cache locality, and workload patterns.

Quick Start Guide

# Write to USB
sudo dd if=xoleric-pc-v1.0.0.iso of=/dev/sdX bs=4M

# Boot from USB
# Select XOLERIC from GRUB menu

# After installation, access AI:
curl -X POST http://localhost:11434/api/chat \
  -d '{"model":"tinyllama","messages":[{"role":"user","content":"Hello!"}]}'

Package Management

# Install package
xpkg install package_name.xpkg

# Remove package  
xpkg remove package_name

# Search packages
xpkg search keyword

# Update all
xpkg update
